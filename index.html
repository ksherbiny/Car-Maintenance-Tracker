Google Sheets Sync

📖 How to set up Google Sheets sync
  1. Go to sheets.google.com → create a new blank sheet, name it Car Maintenance Data
  2. Inside the sheet: Extensions → Apps Script → delete the default code
  3. Copy the script from apps-script.gs in the project → paste it → Save
  4. Click Deploy → New deployment → type: Web app → Execute as: Me → Access: Anyone → Deploy
  5. Authorize access when prompted → copy the Web App URL
  6. Paste that URL in the field above → tap Connect Google Sheets
